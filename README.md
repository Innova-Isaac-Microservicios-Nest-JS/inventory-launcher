# Microservicios Innova Code

## Guia de instalación
Esta guía te ayuda a instalar cada microservicio y api gateway

1. Instalar proyecto
   ```bash
   $  npm install
   ```
2. Crear servicio de NATS server
    ```bash
   $  docker compose up -d
   ```
3. Ejecutar proyectos ejecutando el comando en cada proyecto
    ```bash
   $  npm run start:dev
   ```
